Détection de clusters infectieux

Démo pédagogique : un script Python qui repère les regroupements anormaux de cas dans des données de laboratoire. Tout s'exécute directement dans ton navigateur.

Comment ça marche ?

Les données ci-dessous sont entièrement inventées. Le script regroupe les résultats par service et par germe, puis vérifie si plusieurs cas surviennent dans une fenêtre de temps courte. Tu peux modifier le seuil et la fenêtre pour voir comment les alertes changent.
